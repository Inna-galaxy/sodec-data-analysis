# SODEC Data Analysis Project

## Project Overview
This project involves cleaning and visualizing publicly available data from
SODEC's annual reports to create a user-friendly tool for artists and
companies with limited resources.

The goal is to transform raw cultural funding data into clear insights
that help stakeholders understand funding trends and opportunities.

## Tools Used
- Excel
- Tableau
- Data Cleaning
- Data Visualization

## Process
1. Collected publicly available SODEC annual report data
2. Cleaned and standardized datasets
3. Combined multiple datasets into a master dataset
4. Built visualizations to highlight trends and funding patterns
5. Created an interactive dashboard

## Key Insights
- Identified trends in cultural funding over multiple years
- Highlighted funding distribution across programs
- Provided a simple dashboard for data exploration
